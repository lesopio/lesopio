## Abstract

I am **lesopio / bluebattery**, currently interested in **G-RAG**, **knowledge base construction**, **LLM applications**, and the transformation of unstructured information into usable knowledge.

My work is mostly about small but useful systems: parsing documents, cleaning data, building domain knowledge bases, testing LLM workflows, and turning scattered text into structured formats.
