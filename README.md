## Abstract / 简介

我是 **lesopio / bluebattery**，目前关注 **G-RAG、知识库构建、LLM 应用、信息结构化** 等方向。

I am **lesopio / bluebattery**, currently interested in **G-RAG, knowledge base construction, LLM applications, and information structuring**.

我的兴趣主要集中在：把零散、混乱、非结构化的信息，整理成可以被检索、分析、复用和进一步建模的知识。

My interest lies in transforming scattered and unstructured information into reusable knowledge that can be searched, analyzed, and modeled.

## Selected Projects / 项目展示

### [`TCMdoc_to_json`](https://github.com/lesopio/TCMdoc_to_json)

**中文：**  
面向中医药文档的结构化处理与知识库构建实验。尝试结合 LLM 与 Agent，把原始文本转化为更适合知识库、RAG 和后续模型处理的 JSON / 结构化数据。

**English:**  
An experimental project for structuring TCM documents and building knowledge bases. It explores the use of LLMs and agents to transform raw text into structured data for knowledge bases, RAG workflows, and downstream modeling.

`LLM` `Agent` `TCM` `Knowledge Base` `JSON`

---

### [`wjx_to_files`](https://github.com/lesopio/wjx_to_files)

**中文：**  
问卷页面解析与多格式导出工具。支持将问卷内容解析为 DOCX、JSON 和 Markdown，便于阅读、存档、数据清洗和大模型上下文输入。

**English:**  
A questionnaire parsing tool that converts questionnaire pages into DOCX, JSON, and Markdown formats for reading, archiving, data cleaning, and LLM context input.

`Python` `Parser` `DOCX` `JSON` `Markdown`

---

### [`Collect-questionnaires-from-your-AI`](https://github.com/lesopio/Collect-questionnaires-from-your-AI)

**中文：**  
授权问卷流程自动化测试与 AI 辅助样本生成实验。用于探索 LLM、Playwright 和轻量前端在问卷流程测试中的结合方式。

**English:**  
An experimental project for authorized questionnaire workflow testing and LLM-assisted sample generation, combining LLM APIs, Playwright, and lightweight frontend interfaces.

`Playwright` `LLM API` `Gradio` `Automation`

---

### [`airi_for_todo`](https://github.com/lesopio/airi_for_todo)

**中文：**  
个人 AI / Todo / 数字生活方向实验。用于探索 AI 工具如何嵌入个人任务管理和日常信息处理流程。

**English:**  
A personal AI / Todo / digital-life experiment that explores how AI tools can be integrated into personal task management and everyday information workflows.

`AI Tool` `Todo` `Workflow` `Digital Life`
